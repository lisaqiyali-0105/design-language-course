Challenge 02 / 05

Which form has better spatial rhythm?

Module 04 — Spacing & Layout
A
✓ Consistent rhythm
B
✗ Inconsistent spacing

A has consistent spatial rhythm. Every field uses the same label-to-input gap (5px), the same field-to-field gap (14px), and the same internal padding (8px). In B, the spacing is arbitrary — 1px label gap in the first field, 9px in the second; the button is nearly flat. Spatial consistency is invisible when done right and immediately jarring when wrong.

Challenge 03 / 05

Which passage is easier to read?

Module 02 — Typography
A
THE IMPORTANCE OF GOOD TYPOGRAPHY IN USER INTERFACES
GOOD TYPOGRAPHY IS INVISIBLE. WHEN TYPE WORKS, READERS ABSORB CONTENT WITHOUT NOTICING THE VESSEL. WHEN IT FAILS, THE FRICTION IS FELT BEFORE IT IS NAMED.
✗ All-caps body text
B
The importance of good typography
Good typography is invisible. When type works, readers absorb content without noticing the vessel. When it fails, the friction is felt before it is named.
✓ Proper hierarchy

B is dramatically easier to read. Option A uses all-caps for body copy — a common mistake. All-caps works for 2–4 word labels; it destroys legibility at paragraph length because the eye reads the silhouette of word shapes, not individual letters. All-caps flattens those shapes into uniform rectangles.

B

BETA

✗ Accent color diluted

A uses color as a signal, not decoration. In B, both the "BETA" badge and the CTA button use the same accent green — so the color no longer communicates priority, it just means "thing that exists." Accent colors lose meaning when overused. Every additional use dilutes the signal.

Challenge 05 / 05

Which notification is better aligned?

Module 06 — Gestalt
A
Your report is ready
We've finished processing your Q4 export. Download it now or view the summary in the dashboard.
✗ Mixed alignment axes
B
Your report is ready
We've finished processing your Q4 export. Download it now or view the summary in the dashboard.
✓ Consistent left axis

B has a single, consistent alignment axis. In A, the heading is centered, the body is left-aligned, and the button is centered again — three elements pulling in two directions. Option B aligns everything to the left edge. Picking one alignment axis and committing to it is a fundamental principle of visual coherence.
